# CHALLENGE1AluraStoreLatam
#  Análisis de Tiendas - Alura Store

Este proyecto consiste en un análisis de datos de ventas de las tiendas de la cadena **Alura Store** con el objetivo de identificar cuál de ellas presenta el menor desempeño y debería venderse para iniciar un nuevo emprendimiento.

El análisis se realizó utilizando **Python**, **Pandas** y **Matplotlib**, evaluando diferentes indicadores clave del negocio como ingresos, categorías más vendidas, calificaciones de clientes y costos de envío.

---

#  Objetivo del Proyecto

Ayudar al Sr. Juan a tomar una decisión estratégica sobre cuál de sus tiendas vender, basándose en un análisis de datos que permita comparar el rendimiento de cada una.

---

#  Tecnologías Utilizadas

- Python
- Pandas
- Matplotlib
- Google Colab / Jupyter Notebook

---

#  Dataset

Los datos utilizados contienen información sobre las ventas de **cuatro tiendas** e incluyen variables como:

- Producto
- Categoría del producto
- Precio
- Costo de envío
- Fecha de compra
- Vendedor
- Lugar de compra
- Calificación del cliente
- Método de pago
- Número de cuotas
- Coordenadas geográficas

Cada tienda se encuentra almacenada en un archivo CSV independiente.

---

#  Análisis Realizados

Durante el proyecto se realizaron los siguientes análisis:

###  Facturación total por tienda
Se calculó el total de ingresos generados por cada tienda para identificar cuál produce mayores y menores ganancias.

###  Ventas por categoría
Se analizaron las categorías de productos más vendidas para entender el comportamiento de los clientes.

###  Calificación promedio de clientes
Se evaluó la satisfacción de los clientes mediante el promedio de calificaciones por tienda.

###  Productos más vendidos
Se identificaron los productos con mayor número de ventas en cada tienda.

###  Costo de envío promedio
Se calculó el costo promedio de envío para evaluar posibles impactos en la experiencia del cliente.

---

#  Visualizaciones

Para facilitar la interpretación de los datos se generaron distintos gráficos utilizando **Matplotlib**, entre ellos:

- Gráfico de barras de facturación por tienda
- Gráfico de ventas por categoría
- Gráfico de calificación promedio
- Gráficos de productos más vendidos
- Gráfico circular de costos de envío

---

#  Resultados

Después del análisis se observó que **la Tienda 4 presenta el desempeño más bajo** en comparación con las demás tiendas.

Las principales razones son:

- Es la tienda con **menor facturación total**
- No lidera en ninguna categoría de ventas
- Su rendimiento general es inferior al de las otras tiendas

---

#  Conclusión

Con base en el análisis de datos realizado, se recomienda que **el Sr. Juan venda la Tienda 4**, ya que presenta el menor rendimiento dentro de la cadena y su venta permitiría enfocar recursos en las tiendas con mayor potencial de crecimiento.
